Здесь собран код, при помощи которого происходила обработка данных с эксперимента PAMELA (lab1) и двух различных детекторов (НИРС). 
Пара моментов: в lab1 - было построено множество графиков, вид которых объясняется физическими законами. В НИРС представлен статистический анализ рассчитанной мною МДА (минимальная детектируемая активность) для двух детекторов. При помощи теста Манна-Уитни (предполагает ранжирование, устойчив к выбросам) было получено p-value, меньше уровня значимости 0.05, есть статистически значимые различия, которые зависят от вида выбранного детектора. Исходя из всех результатов, в научной работе были сделаны определенные выводы.
